On windows, the Age of Empires 2 DE folder should be moved to the path:

C:\Users\your_username\Games\

This folder only contains a clean save profile, so your own saved games and mods should be unaltered.	

The scenarios were designed to reproduce the experimental task used in the study, where participants had to complete a primary objective while reacting to four attacks on their base.

Each mod replaces the original “base under attack” notification with the respective auditory notification used in the experiment. Only one of these three mods should be active at a time, so that the correct Auditory Notification with the LOWEST(!!!!) priority, i.e. the last one to be loaded, is used during each experimental condition.
